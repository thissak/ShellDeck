## 다음 작업 (우선순위순)

### 1. Mosh UDP 검증 (블로커: 네트워크)
- alfred 라우터에서 UDP 60000-61000 → 192.168.0.100 포트포워딩 추가
- macOS 방화벽: mosh-server 허용 완료, 스텔스 모드 OFF 완료
- 포트포워딩만 되면 `mosh_main`이 이미 동작하므로 바로 검증 가능
- **확인 파일**: `ShellDeck/Services/MoshSession.swift`

### 2. 한국어 IME 수정
- 증상: iOS 소프트웨어 키보드 한국어 입력 시 글자 깨짐/중복
- 원인 추정: iOS 자동수정/자동대문자/스마트 따옴표가 터미널 입력 변환
- 조사 대상: SwiftTerm `iOSTerminalView.swift`의 UITextInputTraits 설정
- Blink Shell은 WKWebView + ObjC 스위즐링으로 해결 (과잉일 수 있음)
- **확인 파일**: SwiftTerm 소스 `Sources/SwiftTerm/iOS/iOSTerminalView.swift`

### 3. 호스트 편집 UI
- 현재 추가/삭제만 가능, 수정 불가
- HostFormView를 편집 모드로 재사용 (기존 값 프리필)
- **수정 파일**: `ShellDeck/Views/HostList/HostListView.swift`, `HostFormView.swift`

### 4. Host Key Verification (TOFU)
- 현재 `.acceptAnything()` — 모든 호스트 키를 무조건 수락
- HostKeyService는 구현 완료 (verify/trust/remove)
- SSH 연결 시 서버 키를 HostKeyService와 대조, 최초 연결 시 "Trust?" 다이얼로그
- **수정 파일**: `ShellDeck/Views/Terminal/TerminalRepresentable.swift`

### 5. 실기기(iPhone) 배포
- Apple Developer 인증서 필요
- Xcode에서 Signing & Capabilities 설정
- mosh.xcframework가 실기기 ARM64에서 동작하는지 확인

### 6. iCloud 동기화
- HostStorage가 JSON 파일 기반 — NSUbiquitousKeyValueStore 또는 CloudKit으로 확장
- 프라이빗 키/비밀번호는 Keychain에만 저장 (동기화 안 함)
- **수정 파일**: `ShellDeck/Services/HostStorage.swift`

### 7. SFTP (v1.1)
- Citadel SFTP API 조사 필요
- 파일 브라우저 UI
- 새 화면 또는 터미널 내 모드 전환
